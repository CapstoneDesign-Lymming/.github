## 과제 목적 👍

프로젝트 팀원을 구하는 기존 방법은 크게 두 가지로 나뉩니다. 첫째, 인맥을 활용하는 방법, 둘째, 커뮤니티를 통해 모집하는 방식입
니다. 하지만 인맥이 부족하면 팀원을 찾기 어렵고, 커뮤니티에서는 개발자의 정보나 기술 스택을 명확히 파악하기 어렵다는 한계가
있습니다. 이러한 문제를 해결하고자, 사용자가 원하는 스펙의 팀원을 손쉽게 찾을 수 있는 웹 서비스 리밍을 기획하게 되었습니다.
